Flybondi · Anunciador Multilingüe (PWA)
===========================================

Novedades
---------
- Selección de **saludo**: Buenos días / Buenas tardes / Buenas noches.
- Nuevas plantillas: **Demora**, **Cancelación**, **Vuelo condicional** (con **aeropuerto alternativo**).
- Llamada por **múltiples grupos** con indicación de **filas por grupo** y traducción ES/EN/PT.

Uso
---
1) Abrí `index.html` con un servidor local (VS/VS Code).
2) Tocá **“Instalar app”** para guardarla como PWA.
3) Completá **N° de vuelo**, **Destino**, elegí **plantilla**, **saludo** y (si corresponde) **alternativo**.
4) Ajustá voces, tono, velocidad y volumen; reproducí el anuncio.

Destinos
--------
Editá `data/destinations.json` (lista ordenada alfabéticamente).

Notas de voz
------------
Usa Web Speech del dispositivo. Para voces neurales (OpenAI/Azure/GCP), se puede integrar a pedido.
